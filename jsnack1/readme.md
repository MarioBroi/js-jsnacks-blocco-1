# Instructions
- L’utente inserisce due numeri in successione, con due prompt.
- Il software stampa il maggiore.

## Steps
- Chieder 2 numeri tramite prompt
- IF numero1 é MAGGIORE di numero2
    - stampare in console numero1
- ELSE
    - stampare numero2

### Tools
- const / let
- prompt
- if / else
- console.log
